Etiquetas Semanticas

Las etiquetas semanticas consiste en la utilizacion de una serie de, hasta la presente, 84 etiquetas. Cada etiqueta semantica es una balise informatica que, una vez aplicada, permite describir las caracteristicas de tipo y genero textuales. Las etiquetas pertenecen tanto a la microestructura como a la macroestructura.
Las etiquetas semanticas ayudan a la implementacion de los estandares de la Web Semantica, cuya finalidad es hacer el contenido de internet legible para aplicaciones informaticas.
Uno de los usos mas importantes de este tipo de etiquetas es la de marcar la informacion de la pagina para ser indexada por los buscadores web como Google.
Dentro del etiquetado semantico tambien tenemos varias funciones, pero para los que se dedican a maquetar paginas web, hay un conjunto de ellas que nos interesan especialmente. Son las que sirven para definir el esquema principal del documento, como ENCABEZADO, ARTICULO, PIE DE PAGINA, etc. Todas esas etiquetas semanticas nos indican que es el contenido que engloban y cual es su relacion con el conjunto de elementos del documento HTML.

Zanotti, M. S. (2015). Accesibilidad y crowdsourcing: uso de etiquetas semánticas para mejorar la accesibilidad de aplicaciones web (Doctoral dissertation, Universidad Nacional de La Plata).

CSS Grid

Al crear un sitio web, gran parte del trabajo consiste en disponer correctamente los distintos elementos que la componen. El diseño debe ser atractivo, pero claro al mismo tiempo, y se debe comprender de forma intuitiva. Las hojas de estilo en cascada CSS constituyen una herramienta para diseñar sitios web con esas caracteristicas. Mientras que HTML muestra el contenido de forma rudimentaria, CSS es apto para diseños complejos. La tecnología de la web esta en constante desarrollo y ahora, con CSS3, han aparecido técnicas nuevas que permiten aplicar el lenguaje en el internet móvil y utilizar el diseño responsivo.
El CSS Grid se puede utilizar para lograr muchos diseños diferentes. Tambien se destaca por permitir dividir una pagina en areas o regiones principales, por definir la relacion en terminos de tamaño, posicion y capas entre partes de un control constituido a partir de primitivas HTML.
Al igual que las tablas, el Grid layout permite a un autor alinear elementos en columnas y filas. Sin embargo, con CSS Grid son posibles muchos mas diseños y de formas mas sencillas que con las tablas. Por ejemplo, los elementos secundarios de un contenedor de cuadricula podrian posicionarse para que se solapen y se superpongan, de forma similar a los elementos posicionados en CSS.

Seydametov, G.S. y Biktimirov, R.R. (2018). Uso de diseño de cuadrícula css al construir cuadrículas de sitios. Tecnologías de la información e informática en la esfera económica, educativa y social , (3), 51-58.
