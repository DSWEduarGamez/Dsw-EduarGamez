  • Introduccion a la programacion

    • Las aplicaciones web surgieron con la evolución que ha tenido el internet; la construcción de este tipo de aplicaciones se encuentran diseñadas para ser expuestas y desplegadas en internet y de esta manera puedan estar disponibles para cientos de usuarios alrededor del mundo. La programación web nos permite construir este tipo de aplicaciones teniendo claridad en la diferencia entre 3 conceptos clave, diseño web enfocado en la maquetación y embellecimiento de la página, programación web orientada a dar funcionalidad al diseño generado y finalmente el desarrollo web orientado a la integración de la programación más la manipulación de información.
